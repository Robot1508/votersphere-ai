# 🗳️ VoterSphere AI

VoterSphere AI is an interactive platform designed to educate and engage users about the election process through gamification and AI. The project aims to make civic awareness simple, engaging, and accessible using modern web technologies.

---

## 🚀 Live Demo
https://votersphere-ai.vercel.app

---

## 📌 Problem Statement

Many citizens, especially first-time voters, lack awareness and understanding of the election process. Traditional methods of education are often passive and unengaging.

---

## 💡 Solution

VoterSphere AI provides a gamified and interactive experience to help users:
- Understand the voting process
- Learn about EVM functionality
- Explore civic responsibilities
- Get AI-powered answers related to the Constitution

---

## ✨ Features

- 📊 **Interactive Dashboard**  
  Tracks user progress and engagement

- 🎮 **Gamified Voter Journey**  
  Step-by-step simulation of the voting process

- 🗳️ **EVM Simulation**  
  Helps users understand how electronic voting works

- 🤖 **Ask Constitution (AI Assistant)**  
  Users can ask questions related to the Constitution and receive AI-generated answers

---

## 🧠 Google Services Used

- **Google Gemini API**  
  Used to power the AI-based “Ask Constitution” feature for generating intelligent and contextual responses.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Node.js, API Routes  
- **Database:** Supabase (PostgreSQL)  
- **AI Integration:** Google Gemini API  
- **Deployment:** Vercel  

---
